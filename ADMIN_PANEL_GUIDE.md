## Getting Started

### Accessing the Admin Panel
1. Navigate to: `http://your-domain.com/admin/dashboard`
2. Login with your admin credentials
3. You'll see the admin dashboard

### Admin Dashboard
The dashboard shows:
- Quick statistics (users, bookings, inquiries)
- Quick action buttons
- Content management section
- Recent activity feed

## Managing Products

### Viewing Products
1. Click "Products" in Content Management
2. See list of all products with:
   - Name
   - Slug (URL)
   - Category
   - Pricing
   - Status
   - Actions (Edit/Delete)

### Adding a New Product
1. Click "Add Product" button
2. Fill in required fields:
   - **Name**: Product name (auto-generates slug)
   - **Slug**: URL-friendly identifier
   - **Description**: Short description (shown in cards)
   - **Long Description**: Detailed description (shown on product page)
   - **Pricing Badge**: e.g., "From $99/mo" or "Custom Pricing"
   - **Category**: Analytics, Imagery, Data, or Plugin
   - **Status**: Active (visible) or Inactive (hidden)
   - **Order**: Display order (lower numbers first)

3. Upload product image:
   - Click upload area
   - Select image (PNG, JPG, GIF)
   - Max size: 5MB
   - Preview appears after upload

4. Add Features (optional):
   - Click "Add Feature"
   - Enter title, description, icon name
   - Repeat for multiple features
   - Remove with trash icon

5. Add Use Cases (optional):
   - Click "Add Use Case"
   - Enter title, description, industry
   - Repeat as needed

6. Add Specifications (optional):
   - Click "Add Specification"
   - Enter key, value, unit
   - Example: "Resolution", "30cm", ""

7. Click "Create Product"

### Editing a Product
1. Click Edit icon (pencil) on product row
2. Modify any fields
3. Click "Update Product"

### Deleting a Product
1. Click Delete icon (trash) on product row
2. Confirm deletion
3. Product is permanently removed

### Product Best Practices
- Use clear, descriptive names
- Keep slugs URL-friendly (lowercase, hyphens)
- Write compelling descriptions
- Upload high-quality images
- Add at least 2-3 features
- Include relevant use cases
- Set appropriate status (active for live)

## Managing Blog Posts

### Viewing Blog Posts
1. Click "Blog Posts" in Content Management
2. See list with:
   - Title
   - Author
   - Tags
   - Status (Draft/Published)
   - Published date
   - Actions

### Adding a New Blog Post
1. Click "Add Blog Post" button
2. Fill in fields:
   - **Title**: Post title
   - **Slug**: URL identifier
   - **Excerpt**: Short summary (shown in cards)
   - **Content**: Full post content (supports Markdown)
   - **Status**: Draft or Published

3. Add Tags:
   - Type tag name
   - Press Enter or click Add
   - Remove with X icon
   - Use relevant keywords

4. Upload featured image:
   - Landscape orientation recommended
   - High resolution (1200x630 ideal)

5. Click "Create Post" (saves as draft) or set status to Published

### Blog Post Workflow
1. Create post as Draft
2. Write and format content
3. Add tags and image
4. Preview (optional)
5. Change status to Published
6. Post goes live immediately

### Blog Best Practices
- Write compelling titles
- Use clear, concise excerpts
- Format content with headings
- Add relevant tags (3-5 recommended)
- Use high-quality featured images
- Proofread before publishing
- Start as draft, publish when ready

## Image Management

### Uploading Images
1. Click upload area in any form
2. Select image file
3. Wait for upload (progress shown)
4. Preview appears when complete

### Image Requirements
- **Formats**: PNG, JPG, GIF
- **Max Size**: 5MB
- **Recommended**: High resolution, optimized for web

### Image Best Practices
- Use descriptive filenames
- Optimize before upload (compress)
- Use appropriate dimensions
- Test on different devices
- Replace placeholder images

### Removing Images
1. Click X button on image preview
2. Upload new image or leave empty
3. Save form to apply changes

## Troubleshooting

### Can't Login
- Verify credentials
- Check admin role
- Clear browser cache
- Try different browser

### Image Won't Upload
- Check file size (max 5MB)
- Verify file format (PNG, JPG, GIF)
- Try different image
- Check internet connection

### Changes Not Showing
- Refresh browser
- Clear cache
- Check status (must be Active/Published)
- Verify save was successful

### Form Won't Submit
- Check required fields (marked with *)
- Verify slug is unique
- Check for error messages
- Try again

### Content Not Appearing on Site
- Verify status is Active/Published
- Check order number
- Refresh frontend
- Clear browser cache

### Getting Errors
- Note error message
- Check browser console
- Verify all required fields
- Contact support if persists

## Quick Reference

### Keyboard Shortcuts
- `Ctrl/Cmd + S`: Save (in forms)
- `Esc`: Cancel/Close
- `Tab`: Next field
- `Shift + Tab`: Previous field

### Status Meanings
- **Active**: Visible on website
- **Inactive**: Hidden from website
- **Draft**: Not published
- **Published**: Live on website

### Common Actions
- **Create**: Add new content
- **Edit**: Modify existing content
